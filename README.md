# 2D Unity Game

A 2D game built in Unity using C#. This project focuses on gameplay architecture, procedural or dynamic systems, and real-time interaction handling.

## Features

- 2D gameplay systems
- Player movement and interaction
- Collision detection
- Dynamic gameplay behavior
- State-driven logic

## Tech Stack

- Unity
- C#
- 2D Physics
- Procedural Systems
- UI / Interaction Systems

## What I Worked On

- Developed gameplay systems using Unity and C#
- Implemented real-time interaction logic and collision handling
- Built state-based systems to support consistent gameplay flow
- Debugged runtime issues affecting synchronization and responsiveness

## How to Run

1. Clone the repository
2. Open the project in Unity
3. Load the main scene
4. Press Play in the Unity Editor

## Project Focus

This project helped me practice:
- gameplay system design
- state management
- debugging real-time systems
- building interactive applications in Unity
